# 修仙模拟器 (Cultivation Simulator)

一个基于文本的修仙主题桌面游戏，融合了传统的修仙元素和现代的游戏机制。

## 🎮 游戏特色

- **沉浸式修仙体验**：从凡人开始，逐步修炼至渡劫飞升
- **AI世界模拟**：动态生成的世界事件和NPC行为
- **丰富的修炼体系**：多种修炼方式和技能系统
- **策略性资源管理**：合理分配属性点和管理资源

## 🚀 快速开始

### 运行游戏
```bash
# 方法1：直接运行Python文件
python cultivation_game.py

# 方法2：使用启动脚本（Windows）
start_game.bat
```

### 系统要求
- Python 3.8+
- 基础的命令行操作能力

## 🎯 游戏玩法

### 核心机制
1. **属性系统**：
   - 体质：影响生命值和恢复速度
   - 灵根：影响灵气吸收效率
   - 悟性：影响学习和领悟速度
   - 机缘：影响奇遇概率

2. **境界系统**：
   凡人 → 练气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 合体期 → 大乘期 → 渡劫期

3. **资源管理**：
   - 灵石：基础货币
   - 灵药：炼丹材料
   - 法器：装备道具
   - 丹药：消耗品

### 主要功能
- 🔥 **修炼**：提升修为，突破境界
- 🔍 **探索**：发现新地点和奇遇
- 💊 **炼丹**：制作增强属性的丹药
- ⚔️ **战斗**：与其他修士切磋
- 🤝 **社交**：与NPC互动建立关系
