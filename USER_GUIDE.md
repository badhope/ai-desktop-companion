# 🤖 AI桌面伴侣 用户指南

## 🚀 快速开始

### 方式一：双击启动（推荐）
直接双击 `start_ai_companion.bat` 文件，按照提示选择启动模式。

### 方式二：命令行启动
```bash
# 智能启动（自动选择最佳模式）
python smart_starter.py

# 快速启动（基础模式）
python quick_start.py

# 系统检查
python system_checker.py
```

## 🎯 启动模式说明

### 1. 智能启动模式 ⭐（推荐）
- 自动检测系统环境
- 智能选择最佳运行模式
- 自动安装缺失依赖
- 优雅降级处理

### 2. 基础模式
- 最小化依赖要求
- 基础桌面伴侣功能
- 适合配置较低的系统

### 3. 标准模式
- 完整桌面伴侣功能
- 需要基本依赖包
- 平衡性能和功能

### 4. 增强模式
- AI完整功能集成
- 需要所有依赖包
- 功能最丰富但要求最高

## 🔧 系统要求

### 最低配置
- Windows 10/11
- Python 3.8+
- 2GB RAM
- 基础依赖包

### 推荐配置
- Windows 11
- Python 3.10+
- 4GB+ RAM
- 完整依赖包

- `pygame` - 音频和游戏功能

## ⚙️ 配置说明

### 配置文件位置
- `config.py` - 主配置文件
- `starter_config.json` - 启动器配置

### 主要配置项
```python
# 应用基本信息
APP_NAME = "AI桌面伴侣"
VERSION = "3.0"

# 功能开关
ADMIN_REQUIRED = False  # 是否需要管理员权限
SYSTEM_ACCESS = True    # 是否启用系统级访问
AUTO_START = True       # 是否开机自启
```

## 🔧 故障排除

### 常见问题

#### 1. 依赖安装失败
```
解决方案：
- 使用智能启动器自动处理
- 手动安装：pip install -r requirements_optimized.txt
- 跳过可选依赖，使用基础模式运行
```

#### 2. 启动后无界面
```
解决方案：
- 检查Python Tkinter是否安装
- 尝试不同的启动模式
- 查看命令行输出的错误信息
```

#### 3. 功能受限
```
解决方案：
- 运行系统检查工具诊断环境
- 安装缺失的可选依赖
- 切换到更高权限的运行模式
```

### 日志查看
日志文件位于 `logs/` 目录下，可以查看详细的运行信息和错误记录。
