# 妖怪道模拟器 - 用户指南

## 🎮 游戏简介

欢迎来到道士职业模拟器！这是一款深度还原传统修仙文化的文字冒险游戏。在这里，你将体验从凡人到仙人的完整修仙历程。

### 🌟 游戏特色

- **完整修仙体系**：境界突破、功法修炼、法宝收集
- **智能AI引导**：个性化NPC陪伴你的修仙之路
- **丰富剧情任务**：跌宕起伏的修仙故事线
- **真实系统模拟**：炼丹、种植、战斗等完整玩法

## 🚀 快速开始

### 对于普通玩家

**最简单的开始方式：**

1. **双击启动**：直接运行 `start_game.bat`
2. **新手引导**：首次运行会自动进行环境检查
3. **开始游戏**：按照提示创建角色即可开始

**三种启动方式：**

| 方式 | 适用人群 | 特点 |
|------|----------|------|
| `start_game.bat` | 所有人 | 最简单，一键启动 |
| `user_friendly_launcher.py` | 新手推荐 | 有详细提示和帮助 |
| `enhanced_game.py` | 熟练玩家 | 直接进入游戏 |

### 对于开发者

**环境配置：**

```bash
# 1. 克隆项目
git clone <repository-url>
cd ai-desktop-companion

# 2. 运行环境检查
python setup_checker.py

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动游戏
python enhanced_game.py
```

## 🎯 游戏玩法

### 基础操作

1. **角色创建**：分配属性点（体质、灵根、悟性、机缘）
2. **日常修炼**：通过修炼提升修为和境界
3. **资源收集**：探索、任务、种植获得各种资源
4. **系统发展**：炼丹、炼器、学习功法提升实力

### 主要系统

#### 🔥 修炼系统
- **境界突破**：凡人→练气期→筑基期→金丹期→元婴期...
- **修为积累**：通过各种活动获得修为点
- **属性成长**：四大基础属性影响各方面能力

#### ⚗️ 炼丹系统
- **丹药品级**：凡品→灵品→仙品→神品
- **经典丹方**：聚气丹、筑基丹、金元丹等
- **成功率机制**：等级、控火、运气影响成功率

#### 🗡️ 法宝系统
- **法宝等级**：法器→灵器→宝器→仙器→神器
- **四大类别**：武器、防具、辅助、特殊法宝
- **精炼认主**：提升法宝威力和个人绑定

#### 🌱 种植系统
- **灵田管理**：种植各种灵草和仙果
- **生长周期**：真实的种植时间和品质系统
- **农事操作**：浇水、施肥、除草等管理

## ❓ 常见问题

### 环境问题

**Q: 提示Python版本过低怎么办？**
A: 访问 python.org 下载Python 3.8或更高版本，安装时记得勾选"Add Python to PATH"

**Q: 缺少依赖包怎么办？**
A: 运行 `python setup_checker.py` 自动检查并安装缺失的包

**Q: 游戏启动报错怎么办？**
A: 先运行环境检查，确保所有必需组件都已安装

### 游戏问题

**Q: 如何保存游戏进度？**
A: 游戏会自动保存，也可以手动选择"保存游戏"选项

**Q: 存档文件在哪里？**
A: 存档保存在 `saves/` 目录下

**Q: 忘记存档名称怎么办？**
A: 可以查看 `saves/` 目录下的文件名

**Q: 游戏运行很卡怎么办？**
A: 关闭其他程序，确保有足够的内存；降低游戏复杂度

### 操作问题

**Q: 不知道下一步该做什么？**
A: 查看AI引导员的建议，或选择"任务系统"查看当前目标

**Q: 资源不够用怎么办？**
A: 多进行探索和任务，或者使用农场系统种植获取资源

**Q: 战斗总是失败怎么办？**
A: 提升修为境界，获取更强法宝，学习合适功法
